Afterstill is a mindblowing, aesthetic literacy showcase with an intelligent discovery and reading experience.

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open http://localhost:3000 with your browser to see the result.

You can start editing the UI in src/app/page.tsx. The page auto-updates as you edit the file.

## Build

```bash
npm run build
```

## Structure

- src/app/page.tsx: UI/UX layout
- src/app/globals.css: global styles and effects
- src/data/writings.ts: sample writings dataset
- src/lib/analysis.ts: intelligent atlas engine (semantic graph + pathing)
